# 🎬 Netflix Data Analysis Project

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset. The goal is to understand trends in Netflix content such as types of shows, ratings, countries, genres, and content addition patterns over time.

---

## 📊 Dataset Information
- Source: Kaggle (Netflix Movies and TV Shows dataset)
- Total Records: ~8800+
- Features: 12 columns
  - show_id, type, title, director, cast, country, date_added, release_year, rating, duration, listed_in, description

---

## 🎯 Objectives
- Clean and preprocess raw data
- Handle missing values
- Perform exploratory data analysis
- Extract meaningful insights
- Visualize patterns in Netflix content

---

## 🛠️ Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 🧹 Data Cleaning Process
- Handled missing values in columns like director, cast, country
- Removed/handled null values in date_added
- Converted date_added to datetime format
- Created new features like year_added and month_added
- Removed duplicates

---

## 📊 Exploratory Data Analysis (EDA) Insights

- 🎭 Most common genres: **Dramas, International Movies**
- ⭐ Most content is rated: **TV-MA (Mature Audience)**
- 🌍 United States has the highest number of Netflix titles
- 📅 Most content was added in **2021**
- ⏱️ Average movie duration is around **100 minutes**
- 🎬 Netflix has more Movies than TV Shows
